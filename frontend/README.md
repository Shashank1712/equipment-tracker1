# Equipment Tracker

This project is a simple full-stack web application built as a take-home assignment
to demonstrate basic CRUD functionality using React and Node.js.

## Features
- View equipment list
- Add new equipment
- Edit existing equipment
- Delete equipment
- Data persistence using a JSON file

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Storage: JSON file
- Tools: VS Code, Git

## How to Run Locally

### Backend

npm install
node server.js
